# ProjetoPOO

A aplicação deve coletar informações dos Pacientes e classifica-los dentro de 3 categorias:
- Comum
- Grupo Atenção
- Sintoma Covid

Com isso, deve-se encaminhar ao Médico mais adequado, sendo eles:
- Clínico Geral: atendimento de adultos
- Pediatra: atendimento de jovens
- Geriatra: atendimento de idosos
- Infectologista: atendimento de pacientes com sintomas de covid-19

O método Principal está sendo desenvolvido, mas deseja-se que o Paciente tenha uma 
interface amigável, que faça perguntas automáticas, classifique os usuários e encaminhe
para o médico melhor preparado para o atendimento.
